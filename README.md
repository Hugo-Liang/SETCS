# Automatic Code Summarization Using Abbreviation Expansion and Subword Segmentation
This repository contains the datasets and source code for the Semantic Enhanced Transformer for Code Summarization (SETCS) used in our paper entitled "Automatic Code Summarization Using Abbreviation Expansion and Subword Segmentation".


## Datasets
The original (filtered), abbreviation expanded, and tokenized datasets can be found [here](https://drive.google.com/drive/folders/1vjaJGEYyHIVq7ZPq3P_7N_Pc6_q7slS7?usp=sharing). Before running the model, please place the decompressed 'java' subfolder under the 'data' folder.

* Hardwares: Intel(R) Xeon(R) Gold 5118 CPU @ 2.30GHz, 64GB RAM and a Nvidia Tesla P40 GPU with 24GB memory (GPU is a must )
* OS: CentOS 7.6
* Packages:
  * python 3.8
  * pytorch 1.13.0
  * nltk 3.7
